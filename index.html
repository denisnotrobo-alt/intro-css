CSS

Cascading Style Sheets (folha de estilos em cascata), é uma linguagem usada para definir a aparência visual de um site.

CSS não é uma linguagem de programação. É uma linguagem declarativa de estilos.

O css consegue definir:

  • Cores
  • Fontes
  • Espaçamentos
  • Controle de Layout
  • Responsividade
  • Animações
  • Efeitos visuais
  • Posicionamento de elementos

Formas de trabalhar com o CSS

Existem três formas clássicas de se trabalhar com o CSS:

  • Inline
  • Interno
  • Externo
